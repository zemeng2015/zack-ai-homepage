# Zack Wang AI Engineering Homepage

Personal homepage for Applied AI / LLM Systems positioning.

## Purpose

This site is designed to support a job search for AI-related engineering roles:

- Applied AI Engineer
- LLM Systems Engineer
- AI Platform Engineer
- GenAI Backend Engineer
- RAG / Agentic AI Engineer

## Pages

- `index.html`: single-page portfolio homepage.
- `posts/`: technical blog drafts.
- `assets/`: CSS and lightweight site scripts.

## Local Preview

Open `index.html` directly in a browser, or run a simple static server:

```bash
python -m http.server 8080
```

- GitHub Pages
